# Kubernetes 基本概念

## Kubernetes 能做什么

您可以把 Kubernetes 当做一个：

* 容器平台
* 微服务平台
* 便捷式云平台
* 云操作系统
* ......

## Kubernetes 的灵活性和用户选择

* 不限制部署的应用类型
* 不部署源代码
* ...

Kubernetes 不提供这些功能，Cloud Native 提供，这也是我们使用 Kubernetes 必须要做的。

## Kubernetes 与 K8s

Kubernetes 这个名字来自于希腊语，意思是舵手或飞行员。

K8s 是将 “ubernete” 替换为 “8” 而得到的缩写
