# Pod 策略

## 重启策略（`Pod.spec.restartPolicy`）

支持三种 RestartPolicy：

  * Always：只要退出就重启
  * OnFailure：失败退出（exit code 不等于 0）时重启
  * Never：只要退出就不再重启

| 重启策略    | 描述                                                                                                        |
| ----------- | ----------------------------------------------------------------------------------------------------------- |
| `Always`    | 只要退出就重启                                                                                              |
| `OnFailure` | 失败退出时重启；取决于 livenessPorbe 的返回值，如果没有指定 livenessPorbe，将判断 exit code != 0 则认为失败 |
| `Never`     | 退出就不再重启                                                                                              |

注意，这里的重启是指在 Pod 所在 Node 上面本地重启，并不会调度到其他 Node 上去。
